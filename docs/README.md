# Textf Playground

An interactive playground app showcasing all features of the **textf** package — a markdown-like inline text formatting widget for Flutter.

## Features Demonstrated

- **Basic & Advanced Formatting** — Bold, italic, code, strikethrough, underline, highlight, super/subscript
- **Links & Widgets** — Interactive links and inline widget placeholders
- **Styling Options** — Customizable colors, fonts, and text styles
- **Real-World Examples** — Practical use cases for formatted text
- **TextfEditingController** — Live formatting in editable text fields with marker visibility control
- **Theme Support** — Light and dark mode themes

## Getting Started

```bash
flutter run
```

Use the navigation tabs to explore different formatting features and see them rendered in real-time.
